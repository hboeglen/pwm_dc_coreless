Commande PWM de moteur DC coreless avec carte L476RG-Nucleo. 
ADC1 IN1 (PC0 Arduino A5) reliée à un potentiomètre. 
Sortie moteur TIM3 CHANNEL_1 (PA6 Arduino D12). A connecter à un MOSFET de puissance niveau logique (exemple SPP 80N03L). On peut utiliser un module MP MOS Module (https://www.otronic.nl/fr/module-pilote-mos-fet-irf520.html).
